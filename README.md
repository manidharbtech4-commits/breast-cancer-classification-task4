# AI & ML Task 4: Breast Cancer Classification

## Overview
Built a robust Binary Classification system for Breast Cancer diagnosis using scikit-learn.

## Key Features
- Used Breast Cancer Wisconsin Dataset
- Trained Logistic Regression + Decision Tree
- Evaluated using Confusion Matrix, Precision, Recall, F1, ROC-AUC
- Handled potential class imbalance using class weights

## Results
- High AUC Score (~0.99)
- Strong Recall for Malignant class
